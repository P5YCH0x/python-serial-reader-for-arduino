# python-serial-reader-for-arduino
Method of reading arduino serial from python

## This project uses:
  1x arduino,
  1x push button,
  1x 10k resistor,
  5x jumper wires
  
  python3,
  pyserial library(https://pypi.org/project/pyserial/)
